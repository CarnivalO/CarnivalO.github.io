3.一些必要的软件

1.screen

现在ssh远程连接上去最好再用screen建一个会话,避免ssh断开造成会话结束.
安装:apt install screen

2.vim

dietpi没有默认装vim..太精简了..
安装:apt install vim
创建编辑配置文件:vim ~/.vimrc

3.aria2

下载工作交给低功耗电脑最好了,现在还是推荐买个路由刷pandavan,用路由下载更方便~
安装:apt install aria2
在家目录创建.aria2文件夹,在里面创建aria2.conf和aria2.session文件
aria2.conf系统配置
使用screen启动,避免会话结束后aria2被关闭:screen -dmS aria2 aria2c –conf-path=/root/.aria2/aria2.conf
查询使用状态:netstat -anp|grep 6800

4.zsh

安装:apt install zsh
将终端切换为zsh:chsh -s $(which zsh) //要恢复就将zsh改为bash
安装oh my zsh:sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"
增加插件:
将插件clone到~/.oh-my-zsh/plugins

zsh-syntax-highlighting 提供语法高亮：git clone git://github.com/zsh-users/zsh-syntax-highlighting.git
修改~/.zshrc的：plugins=(git z zsh-syntax-highlighting zsh-autosuggestions)
使生效：source ~/.zshrc
